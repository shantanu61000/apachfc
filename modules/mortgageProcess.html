Disbursement / Booking/ Storage:
Process Stage Activity Responsibility
Documentation
  • Print loan documents and obtain customer signature.
  • Collate all customer documents into the customer file.
  • Collect customer Original Collertal documents.
 Relationship Officer
Branch Quality Check
  • Perform a thorough check on the completeness and quality of the documentation.
  • Allow loan disbursement only if the file clears quality check.
 Credit Manager/ Branch Operations
Disbursement / Collateral Custody
  • All disbursements to happen only post customer authentication.
  • All required documents to be scanned and to be uploaded in myfin.
  • First set of quires to be resolved and case to be booked.
  • OTC to be taken for pending documents.
  • Cheque handover to be done post OTC clearance.
  • Credit Manager
  • Operations Officer
Central QC / File Storage
  • Post disbursement, Files are moved to Head office.
  • Documents are scanned and stored at the central QC for future reference.
  • Originals (secured documents) will be retrieved and sent back to the branch as and when required.
 Central Ops
Servicing / Collection:
Process Stage Activity Responsibility
Repayments
  • Customer Calling to be done before nach is presented.
  • Once the NACH status is received, The branch team is required to start following up with the customer for payment.
 Relationship officer/ CM / BM
Collection
  • Visit the Customer on the due date of collection.
  • In case of non-payment, identify reasons and counsel on benefits of prompt payment.
  • Relationship Officer / Relationship Manager
  • Branch Manager
  • Collections Officer
Loan Restructuring
  • Applicable only if the customer is eligible for loan restructuring.
  • Customers need to travel to the branch to avail this facility.
  • The loan is restructured based on the revised cash flow analysis of the customer.
  • Credit Manager
  • Relationship Manager
  • Branch Manager
  • State Risk Manager
Loan Closure
  • With the payment of the last installment, the loan is closed automatically.
  • Any foreclosure requests, if eligible, are serviced only post approval.
  • The process for return of security documents is initiated only post the closure of the loan.
  • Creddit Manager
  • Branch Manager
